# מחולל מרחבי 360°

פלטפורמה ליצירת סביבות למידה תלת-ממדיות 360° עם נקודות חמות, שמירה בענן וקישורי שיתוף לתלמידים.

## מה כלול

| קובץ | תפקיד |
|------|--------|
| `Code.gs` | שרת Apps Script — שמירה ב-Sheet, העלאה ל-Drive |
| `Index.html` | ממשק המחולל (עורך + תצוגת תלמיד) |
| `appsscript.json` | הגדרות פריסה |

## התקנה — שלב אחר שלב

### 1. צור Google Sheet חדש
פתח [Google Sheets](https://sheets.google.com) וצור גיליון ריק.

### 2. פתח Apps Script
תפריט: **תוספים → Apps Script**

### 3. העתק את הקבצים
- מחק את `Code.gs` הקיים והדבק את `Code.gs` מהפרויקט
- צור קובץ HTML בשם **`Index`** (חייב להיות בדיוק `Index`) והדבק את `Index.html`
- עדכן את `appsscript.json` אם צריך

### 4. הרץ הגדרה ראשונית
בעורך Apps Script:
1. בחר את הפונקציה `setupSheet`
2. לחץ **הרץ** — זה ייצור את גיליון `Projects` ותיקיית Drive `360-Environments`

### 5. פרוס כ-Web App
1. **פרוס → פריסה חדשה → אפליקציית אינטרנט**
2. הפעל כ: **אני (המשתמש)**
3. מי יכול לגשת: **כל אחד**
4. לחץ **פרוס** והעתק את כתובת ה-URL

## איך זה עובד

### יצירת מרחב חדש
פתח את כתובת ה-Web App (ללא פרמטרים) → עצב רקע ונקודות → **שמור בענן וקבל קישור**

### קישורים
- **לתלמידים:** `.../exec?id=XXXXXXXXXX` — צפייה בלבד
- **לעריכה:** `.../exec?id=XXXXXXXXXX&edit=TOKEN` — שמור לעצמך!

### מבנה Google Sheet (גיליון `Projects`)

| עמודה | תוכן |
|--------|------|
| id | מזהה ייחודי |
| title | שם המרחב |
| created_at / updated_at | תאריכים |
| image1_url / image2_url | קישורי Drive לרקעים |
| active_image | סצנה פעילה (1 או 2) |
| hotspots_json | JSON של כל הנקודות |
| edit_token | טוקן לעריכה |
| view_url | קישור לתלמידים |
| edit_url | קישור לעריכה |
| copy_url | קישור להעתקה למורה אחר |

### קישורים בטבלה
לאחר כל שמירה, שלושת הקישורים נשמרים אוטומטית בעמודות 10–12. לפרויקטים ישנים הרץ פעם אחת את `backfillUrls` ב-Apps Script.

### קטלוג מרחבים
- מתוך העורך: כפתור **קטלוג כל המרחבים**
- ישירות: `.../exec?page=catalog`

### העתקת מרחב למורה אחר
קישור העתקה: `.../exec?template=PROJECT_ID` — פותח עותק עם כל הרקע והנקודות, לעריכה ושמירה כפרויקט חדש.

### תיקיית Drive
**לא צריך לתת קישור לתיקייה** — בהרצה ראשונה של `setupSheet` נוצרת אוטומטית תיקייה `360-Environments` ב-Drive שלך.

אם יש לך תיקייה קיימת ואתה רוצה להשתמש בה — הדבק את מזהה התיקייה (מה-URL) בשדה `DRIVE_FOLDER_ID` ב-`Code.gs`.

### כתובת קבועה
המערכת עובדת עם כתובת Web App קבועה (מוגדרת ב-`CONFIG.WEB_APP_URL`). הקישורים לתלמידים ולעריכה משתמשים בכתובת זו + פרמטרים (`?id=...`).

## שיפורים במחולל

- **תווית** — מספר או אות (עד 2 תווים) על כל נקודה
- **צבע** — בחירת צבע לכל נקודה
- **תמונה בנקודה** — העלאה או הדבקה (Ctrl+V), נשמרת ב-Drive
- **הדבקת רקע** — Ctrl+V כשהסיידבר פתוח
- **עריכה חוזרת** — קישור עריכה שומר את כל הנתונים

## גיבוי מקומי

הקובץ `360_interactive_environment.html` בשורש הפרויקט עובד גם ללא Apps Script (כולל ייצוא HTML). שמירה בענן פעילה רק בפריסת Apps Script.

## clasp (אופציונלי)

```bash
npm install -g @google/clasp
clasp login
clasp create --type sheets --title "360 Environments"
clasp push
clasp deploy
```
