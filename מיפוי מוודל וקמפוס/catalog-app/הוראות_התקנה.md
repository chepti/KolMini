# קטלוג קורסי מוודל וקמפוס — הוראות התקנה

אפליקציית Apps Script המחוברת לגיליון Google Sheets. מציגה את הקורסים ככרטיסים,
מאפשרת סינון וחיפוש, ולחיצה על כרטיס פותחת חלונית לעריכת המידע ושמירתו חזרה לגיליון.

הקבצים בתיקייה זו:
- `קורסים_לייבוא.csv` — 201 הקורסים מנוקים, כולל עמודות **פלטפורמה**, **הערות** ו**עודכן**.
- `Code.gs` — צד השרת (קריאה/כתיבה לגיליון).
- `Index.html` — הממשק (כרטיסים, סינון, חלונית עריכה). **הלוגואים מוטמעים בתוכו כ-base64** — אין צורך בקישור חיצוני.
- `logos/` — קבצי הלוגו המקוריים (מוודל, קמפוס, אגף האקדמיזציה).
- `inject_logos.ps1` — סקריפט להטמעת הלוגואים מחדש ל-Index.html אם הם משתנים.

---

## שלב 1 — יצירת הגיליון
1. היכנסו לחשבון Google הנכון (זה שבו תרצו שהגיליון יישב).
2. צרו גיליון חדש: https://sheets.new
3. תפריט **קובץ → ייבוא → העלאה** → בחרו את `קורסים_לייבוא.csv`.
4. בחלון הייבוא: **מיקום ייבוא = החלפת הגיליון הנוכחי**, ולחצו **ייבוא נתונים**.
5. ודאו ששורה 1 היא שורת הכותרות (שם_קורס, תחום_דעת, …, הערות, עודכן).

## שלב 2 — הדבקת הקוד
1. בגיליון: **תוספות (Extensions) → Apps Script**.
2. בעורך שנפתח:
   - בקובץ `Code.gs` — מחקו את התוכן והדביקו את כל התוכן מ-`Code.gs` שבתיקייה.
   - לחצו על **+** ליד "Files" → **HTML** → תנו לו שם בדיוק **`Index`** (בלי סיומת).
   - הדביקו לתוכו את כל התוכן מ-`Index.html`.
3. שמרו (Ctrl+S).

## שלב 3 — פריסה (Deploy)
1. כפתור **Deploy → New deployment**.
2. גלגל השיניים → בחרו **Web app**.
3. הגדרות:
   - **Execute as:** Me (החשבון שלכם).
   - **Who has access:** לבחירתכם — "Anyone with the link" כדי לשתף עם מפמ"רים,
     או "Anyone within [הארגון]" אם רוצים להגביל.
4. **Deploy** → אשרו את ההרשאות (Authorize) בפעם הראשונה.
5. העתיקו את ה-**Web app URL** — זה הקישור לשליחה למפמ"רים.

## הערות חשובות
- **זיהוי עמודות אוטומטי:** הקוד מזהה עמודות לפי שמות הכותרות בשורה 1, לא לפי מיקום.
  אפשר להוסיף/לסדר מחדש עמודות בגיליון בלי לשבור את האפליקציה.
- **שמירה:** כל עריכה בחלונית נכתבת ישירות לשורה המתאימה בגיליון, ועמודת **עודכן**
  מתעדכנת אוטומטית עם תאריך ושעה.
- **עמודת הערות:** שדה חופשי לכל הערה של המפמ"ר.
- **אחרי שינוי קוד:** אם תערכו את `Code.gs`/`Index.html` בעתיד — צריך **Deploy → Manage
  deployments → עריכה → Version: New version** כדי שהקישור הקיים יתעדכן.
- האפליקציה בעברית מלאה (RTL), עם רקע בלובים פסטל, צ'יפים מעוגלים וסינון מהיר.
