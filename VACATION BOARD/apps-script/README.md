# שמירה ב-Google Sheets (Apps Script)

הלוח שומר את המשפחה, הצבעים והאירועים בגיליון Google — כולם רואים את אותו לוח.

## התקנה (פעם אחת)

1. צרו [גיליון Google חדש](https://sheets.new)
2. **Extensions → Apps Script**
3. מחקו את הקוד הקיים, והדביקו את התוכן מ־`Code.gs`
4. שמרו (Ctrl+S)
5. **Deploy → New deployment**
   - סוג: **Web app**
   - Execute as: **Me**
   - Who has access: **Anyone**
6. לחצו **Deploy**, העתיקו את כתובת ה־Web App  
   (נראית כמו `https://script.google.com/macros/s/XXXX/exec`)
7. בלוח החופש לחצו **Sheets**, הדביקו את הכתובת ושמרו

## מה נשמר בגיליון

| לשונית | תוכן |
|--------|------|
| הגדרות | טווח תאריכים, הסתרות |
| ענפים | משפחות |
| אנשים | שמות + צבעים |
| פעילויות | אירועים |

אחרי כל שינוי בלוח — הגיליון מתעדכן אוטומטית (עם השהייה קצרה).

## יומן Google (מנוי חי)

אותה כתובת Web App עם `?format=ics` משמשת כהזנת יומן:

`https://script.google.com/macros/s/XXXX/exec?format=ics`

1. עדכנו את `Code.gs` ופרסמו **Deploy** מחדש (פעם אחת אחרי הוספת ICS)
2. בלוח החופש: **ליומן גוגל** → העתיקו את קישור המנוי
3. בגוגל יומן → יומנים אחרים → **+** → מכתובת URL → הדביקו

גוגל מרענן מנויים מעת לעת (לא מיידי) — השינויים בלוח המשפחתי יופיעו גם ביומן.
